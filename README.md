# Hebrew Architecture Corpus

Cleaned, parsed and analyzed corpus of documents pertaining to architecture and built enviroment in Hebrew.

## Contents
- /txt/ - plain text
- /conllu/ - morphologically parsed texts in CONLLU format
- /jsonl/ - Named entities in JSONL format
- /metadata/ - index and miscellenious metadata
